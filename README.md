# MRI-US Image Fusion for Endometriosis Detection

This repository contains code for comparing PALM and DDFM fusion methods, plus hybrid approaches combining both.

## Structure

- `PALM/` – Original PALM implementation [El Mansouri et al. 2020]
- `DDFM/` – DDFM implementation [Zhao et al. 2023] with modifications
- `scripts/` – Experiment runners and evaluation tools
- `results/` – Organized output images and metrics
